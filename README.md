# Terminal Velocity Simulation

## Overview

This project is a **virtual lab simulation** for the experiment **“Determination of Terminal Velocity of a Spherical Body in Water.”**
It visually demonstrates how a sphere falls through a fluid and eventually reaches **terminal velocity**, where the forces acting on the body balance and the velocity becomes constant.

## Objective

* Understand the concept of **terminal velocity**
* Observe the motion of a **sphere in a viscous fluid**
* Record experimental values using a **virtual observation table**

## Technologies Used

* HTML
* CSS
* JavaScript

## How to Run

1. Clone the repository

```
git clone https://github.com/Pradum-Yadav12/Terminal-Velocity-Simulation.git
```

2. Open the project folder and run

```
index.html
```

in any web browser.
